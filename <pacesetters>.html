Internet Basics

What is the Internet?

A global network that connects millions of computers

How does the Internet work?

Packets travel through routers and servers to reach their destination

What is a Domain Name?

A human-friendly name for a website

DNS and How It Works

It converts domain names into IP addresses

Browsers and How They Work

They request web pages from the internet and display them for users

What are Markup Languages?

Languages used to structure and format web pages

What is Front-End Development?

Creating the part of a website users see and interact with

What is HTTP?

HyperText Transfer Protocol

What is a Browser?

A browser is a program that lets users access the web

How Does the Web Work?

The web is a collection of websites stored on servers
